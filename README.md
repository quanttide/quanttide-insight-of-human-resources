## 核心脉络

### 定价即组织设计

薪酬不是分钱问题，而是定义关系的问题。

- 外部渠道价格是**唯一锚点**——消解"公司欠我"的心态
- 10% vs 20% 不是在定工资，是在问"你选雇佣关系还是创始人心态"
- 比例序列的本质是**按资源稀缺度采购**，不论内外，不看岗位

> 不确定的都先不定了，把唯一确定的外部约束作为锚点，其他都围绕它展开。

### 设计服务于人才现实

定价逻辑的底层假设和公司现实严丝合缝：

- 招不到成熟销售 → 就不设计"激励成熟销售"的方案
- 招的是愿意干的零基础实习生 → 10% 天然成为**成长路径的引导信号**
- 先筛选、后引导 → 利益结构代替管理者去推动人

这不是妥协，是把现实约束变成设计输入。

### 决策的贝叶斯基础

管理者的判断力，本质是贝叶斯更新的直觉能力：

| 管理直觉 | 数学对应 |
|---|---|
| 一次不回复不代表什么，但多次就是信号 | 高权重负面信号大幅更新先验 |
| 不能因为动了一下就放松警惕 | 小幅正面信号仅轻微回调后验 |
| 还要继续观察 | 序贯更新，后验变为下一轮先验 |
| 管理者先出招，看反应再调整 | 不知情方设计筛选机制 |

### 贯穿的暗线：极简主义

- 不定复杂的岗位评估 → 只有一个锚点：渠道 20%
- 不靠流程管人 → 靠利益结构引导
- 不确定的先不定 → 先把确定的锚住

> 处理不确定性不是靠增加复杂度，而是靠找到唯一确定的东西。
